# Devcamper API

> Backend API for Devcamper application, which is a bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update values setting

## Install Dependencies

```
npm install
```

# Run App

```
# Run in a dev mode
npm run dev

# Run in a production mode
npm start
```
